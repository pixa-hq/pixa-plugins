# Pixa plugins

Official Pixa plugin for **Claude**

## What this plugin provides

- **Image and video generation** — Use Pixa to create and edit images and video from natural language inside Claude.
- **Image Editing** — Use Pixa to remove backgrounds, upscale images, and more.
- **Library management** — Search and manage your library of images and video.

## Setup

### Claude Cowork
1. Add the marketplace to Claude

- Select Customize from the Sidebar
- Plugins -> "+ Add Plugin" -> "Create Plugin" -> "Add Marketplace"
- Enter `pixa-hq/pixa-plugins`

2. Install the plugin

- Plugins -> "Browse Plugins" -> "Pixa" -> "Install"

### Claude Code CLI

1. Add the marketplace to Claude

```
/plugin marketplace add pixa-hq/pixa-plugins
```

2. Install the plugin

```
/plugin install pixa@pixa-plugins
```
